# Rusty

# Prerequisites

## Linux

1. Install Rust (stable):

        curl https://sh.rustup.rs -sSf | sh

2. SDL2:

    ### Ubuntu

        sudo apt install libsdl2-dev libsdl2-ttf-dev libsdl2-image-dev libsdl2-mixer-dev

    ### Arch Linux

        sudo pacman -S sdl2 sdl2_ttf sdl2_image sdl2_mixer

## Build

    cargo run
